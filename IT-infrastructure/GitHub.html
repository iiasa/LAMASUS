GitHub is the leading code hosting and development collaboration cloud platform. The platform is owned and managed by Microsoft and provides many services free-of-charge. Most developers and many researchers are already familiar with GitHub. Via GitHub Actions, GitHub can integrate with the rest of the LAMASUS IT infrastructure. GitHub is therefore a good default place for LAMASUS code hosting and collaboration.

The GitHub platform does have limitations. Repositories cannot store large files without paying for the Git LFS service. Privacy of stored data is not guaranteed in a verifiable manner. And not all services are available in the for-free tier or for private repositories. In particular, there is only limited availability of the GitHub Actions service for private repositories.
