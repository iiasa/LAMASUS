Introduction

The LAMASUS IT infrastructure provides a shared platform for deployment of models, data repositories, and web apps. By using a shared platform, the exchange of data and creation of interfaces is supported and enables:

  • Dissemination of data and results to the general public.

  • Collaboration on deliverables that cut across project partners.

  • Access to shared datasets and data stores for models and web apps.

  • Automation of links between models.

  • The hosting and publishing of model and tool code.

  • …

Widely-used modern components were chosen to comprise the IT infrastructure so that LAMASUS can benefit from well-developed IT practices and knowledge.
